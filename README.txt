ได้มีการจัดรูปหน้าผ่าน index.css หลายส่วน
มีจำนวนหน้าทั้งหมด 6 หน้า ได้แก่
1. หน้าหลัก
2. หน้าตะกร้า
3. หน้าสินค้าทั้งหมด
4. หน้าสมัครสมาชิก
5. หน้าเข้าสู่ระบบ
6. หน้าสำหรับแอดมินเพิ่มสินค้า

มีการเก็บสถานะการล๊อกอินผ่าน session
มีการตรวฟอร์มว่าไส่ครบหรือไม่และส่งไปยังส่วน php ได้
มีการอัพโหลดรปภาพสินค้าขึ้นไปบน server 
มีการใช้ DOM ในการทำรูปสไลด์บนหน้าเว็บ

โครงสร้างไฟล์
- assets เป็บส่วนประกอบเว็บ
-- css เก็บไฟล์ css ทั้งหมด
-- font เก็บไฟล์ลักษณะตัวอักษร
-- img เก็บรูป
--- shoes เก็บรูปรองเท้าทั้งหมด สำหรับดึงมาแสดงรูปสินค้า
-- js เก็บไฟล์ javascript
- libs ใส่ส่วนประกอบของหน้าเว็บที่มักจะถูกเรียกใช้บ่อยๆ
+ใส่ไฟล์หน้าเว็บทั่วไป
+ มีไฟล์ config สำหรับปรับข้อมูลจำเป็นต่างๆ

วิธีการรันเมื่อให้นำโฟเดอร์ทั้งหมดข้างบนไปใส่ในโฟลเดอร์ที่ทำงาน 
xampp ใน path htdocs\ แล้วนำโฟลเดอร์ MegamindProject ไปวางข้างใน
เมื่อรัน xampp ก็จะสามารถใช้งานหน้าเว็บได้ผ่านทาง localhost 

วิธีการใส่ฐานข้อมูล ให้เข้าไปที่ http://localhost/phpmyadmin 
แล้วนำไฟล์ sql เข้าไปใส่ในส่วนของการ import โดยตั้งชื่อฐานข้อมูล
